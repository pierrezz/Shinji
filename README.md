# Shinji

A modern WhatsApp bot built with Node.js, featuring AI, moderation, entertainment, utility, and automation commands.

---

# Features

- AI Chat
- Fun Commands
- Music
- Sticker Tools
- Media Downloader
- Utilities
- Group Moderation
- Automation
- Information Commands
- Secure & Fast

---

# Requirements

- Node.js 18 or higher
- Google Chrome or Chromium installed
- Internet connection

---

# Installation

Clone the repository:

```bash
git clone https://github.com/pierrezz/Shinji.git
```

Enter the project folder:

```bash
cd Shinji
```

Install the dependencies:

```bash
npm install
```

---

# Running

Start the bot:

```bash
npm start
```

---

# Example

After connecting the bot, use:

```text
!menu
```

to display all available commands.
